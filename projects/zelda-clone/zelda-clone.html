Zelda-Clone

Apologies if I clickbaited you! You can't actually play the game here.

This clone of the original Zelda was built during one semester by a team of five students at OSU. The game was built on MonoGame and C# from the ground-up, and was used as a learning tool to mature our software development skills.

If you so desire, you can compile and run the game yourself: GitHub repo!

This page will be highlighting interesting game features along with snippets of /code/.
